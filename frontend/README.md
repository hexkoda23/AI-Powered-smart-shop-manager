# Shop Management System - Frontend

Next.js frontend for the AI-Powered Shop Management System.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create a `.env.local` file:
```
NEXT_PUBLIC_API_URL=http://localhost:8000
```

3. Run the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Features

- **Dashboard**: Overview of profits, sales, and stock alerts
- **Record Sale**: Simple form to record daily sales
- **Stock Management**: Track inventory levels and set low-stock thresholds
- **AI Assistant**: Chat with AI to get insights and recommendations

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Recharts (for data visualization)
- Lucide React (icons)
